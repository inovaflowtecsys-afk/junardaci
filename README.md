# Sistema Web — Clínica Juliana Nardaci

Sistema de gestão clínica completo e responsivo para clínicas de estética, desenvolvido com React 18 e Vite.

## 🚀 Tecnologias
- **Frontend**: React 18 + Vite
- **Estilo**: CSS Modules + Variáveis CSS nativas
- **Ícones**: Lucide-React
- **Rotas**: React-Router-Dom v6
- **Máscaras**: React-IMask
- **Banco de Dados**: Supabase (Próxima fase)

## 📦 Instalação
1. Clone o repositório
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run dev`

## 🔑 Acesso (Mock)
- **Admin**: `admin@clinica.com` / `Admin@123`
- **Profissional**: `profissional@clinica.com` / `Prof@123`

## 📂 Estrutura de Pastas
- `src/components`: Componentes reutilizáveis (Modal, Input, etc)
- `src/pages`: Páginas de cada módulo (Clientes, Atendimentos, etc)
- `src/data`: Camada de dados fictícios (mockData.js)
- `src/styles`: Variáveis e estilos globais
